# GramSaarthi Frontend

React frontend for GramSaarthi with Indian-themed UI.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Environment Variables

Create `.env` file:

```env
VITE_API_URL=http://localhost:8000
```

## Features

- Indian-themed UI (Saffron & Green colors)
- Persona-based routing
- Location hierarchy selection
- Real-time chatbot
- Dashboard with charts
- Schemes browser
- File upload (District Admin)
